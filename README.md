# Thesis Project

This is a project I built for my final thesis for the GDES program at OCAD University in Toronto, Canada.

This project utilizes:

- [x] Three JS
- [x] React
- [x] Redux
- [x] Typescript

[View the working project](http://www.therippleeffect.ca)

## Setup

```
$ npm install
```

## Running

```
$ npm start
```

## Linting

```
$ npm run lint
```

## Formatting

```
$ npm run prettier
```

## Build

```
$ npm run build
```
